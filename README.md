# 🩸 Blood Donation Website for LPU University

## 📌 Project Overview

The Blood Donation Website is a web-based application developed specifically for Lovely Professional University (LPU) students, faculty, and staff. The platform helps connect blood donors with recipients during emergency situations by providing a centralized system for donor registration, donor search, and blood request management.

The objective of this project is to make blood donation services easily accessible within the university community and reduce the time required to find suitable blood donors.

---

## 🎯 Problem Statement

During medical emergencies, finding blood donors quickly is often difficult. Students and staff generally rely on personal contacts, social media groups, and messaging applications to locate donors. This process can be slow, unorganized, and unreliable.

The Blood Donation Website addresses this problem by providing a centralized platform where users can register as donors, search for donors based on blood groups, and submit blood requests efficiently.

---

## 🏗️ System Modules

### 1. User Registration Module
Allows users to create an account by providing:
- Name
- Email
- Phone Number
- Blood Group
- Department
- Password

### 2. Login Module
Provides secure authentication for users and administrators.

### 3. Donor Management Module
Stores and manages donor information in the database.

### 4. Donor Search Module
Enables users to search donors based on blood groups.

### 5. Blood Request Module
Allows recipients to submit emergency blood requests.

### 6. Admin Module
Provides administrative control over donors and requests.
